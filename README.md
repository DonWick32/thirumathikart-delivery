# Thirumathikart Delivery App

Flutter Application for Thirumathikart-Delivery


## Flutter

### Requirements

- Java-Version - '12.x'
- Flutter-Version - '2.8.x'
- Dart-Version - '2.12' or above
- Android Studio

### Setup
* Fork and Clone the Repo
    ```
    git clone <YOUR_FORK_URL>
    ```
* Add remote upstream
    ```
    git remote add upstream <MAIN_REPO_URL>
    ```
* Run the following command to enable githooks
    ```
    git config core.hooksPath .githooks
    ```    
* To get all the dependencies listed in the pubspec
    ```
    flutter pub get
    ```
* To run the App
    ```
    ./run.sh .env
